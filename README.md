# sorting-for-humans

Sorts strings the way you actually want them sorted. Instead of:

```
0_data.json
0_thing.json
10_data.json
10_thing.json
15_data.json
17_thing.json
1_data.json
1_thing.json
20_data.json
24_thing.json
25_data.json
2_data.json
2_thing.json
filename_0.py
filename_1.py
filename_2.py
filename_10.py
filename_12.py
filename_20.py
stuff_0_63.json
stuff_1_91.json
stuff_10_40.json
stuff_15_3.json
stuff_2_11.json
stuff_20_99.json
stuff_25_32.json
```

We get:
```
0_data.json
1_data.json
2_data.json
10_data.json
15_data.json
20_data.json
25_data.json
0_thing.json
1_thing.json
2_thing.json
10_thing.json
17_thing.json
24_thing.json
filename_0.py
filename_1.py
filename_2.py
filename_10.py
filename_12.py
filename_20.py
stuff_0_63.json
stuff_1_91.json
stuff_2_11.json
stuff_10_40.json
stuff_15_3.json
stuff_20_99.json
stuff_25_32.json
```
